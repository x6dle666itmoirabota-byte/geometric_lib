# Документация по проекту geometric_lib

## Общее описание решения

Проект `geometric_lib` представляет собой библиотеку на языке Python для вычисления площади и периметра различных геометрических фигур. Библиотека организована в виде пакета, где каждый модуль отвечает за отдельную фигуру.

Текущая структура проекта:

```
geometric_lib/
├── circle.py
├── rectangle.py
├── square.py
└── triangle.py
```

## Описание функций

### Круг (`circle.py`)

*   `area(r)`: Вычисляет площадь круга.
    *   **Пример вызова:** `circle.area(10)`
*   `perimeter(r)`: Вычисляет длину окружности.
    *   **Пример вызова:** `circle.perimeter(10)`

### Прямоугольник (`rectangle.py`)

*   `area(a, b)`: Вычисляет площадь прямоугольника.
    *   **Пример вызова:** `rectangle.area(5, 10)`
*   `perimeter(a, b)`: Вычисляет периметр прямоугольника.
    *   **Пример вызова:** `rectangle.perimeter(5, 10)`

### Квадрат (`square.py`)

*   `area(a)`: Вычисляет площадь квадрата.
    *   **Пример вызова:** `square.area(5)`
*   `perimeter(a)`: Вычисляет периметр квадрата.
    *   **Пример вызова:** `square.perimeter(5)`

### Треугольник (`triangle.py`)

*   `area(a, h)`: Вычисляет площадь треугольника по основанию и высоте.
    *   **Пример вызова:** `triangle.area(10, 5)`
*   `perimeter(a, b, c)`: Вычисляет периметр треугольника по трём сторонам.
    *   **Пример вызова:** `triangle.perimeter(3, 4, 5)`
